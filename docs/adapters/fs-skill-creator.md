# FS Skill Creator (`FsSkillCreator`, `FsSkillWriter`)

Creates new skill directories with scaffolded `SKILL.md` files on the filesystem.

## Two Adapters, One File

Both adapters live in `adapters/src/fs_skill_creator.rs` because they serve complementary roles in skill creation.

### `FsSkillCreator` — Directory + Scaffold

```rust
pub struct FsSkillCreator {
    pub base_dir: PathBuf,   // typically ~/.claude/skills
}
```

#### Port

```rust
impl SkillCreator for FsSkillCreator {
    fn create(&self, name: &str, agents: &[String], tags: &[String])
        -> Result<PathBuf, Box<dyn std::error::Error>>;
}
```

#### Behavior

1. Creates directory `{base_dir}/{name}/`
2. Generates scaffold content via `core::scaffold_skill(name, agents, tags)`
3. Writes `{base_dir}/{name}/SKILL.md` with the scaffold
4. Returns `PathBuf` to the created directory

#### Constructor

```rust
pub fn from_env() -> Self   // $HOME/.claude/skills
pub fn new(base_dir: PathBuf) -> Self
```

### `FsSkillWriter` — Content Writer

```rust
pub struct FsSkillWriter;
```

#### Port

```rust
impl SkillWriter for FsSkillWriter {
    fn write(&self, path: &Path, content: &str)
        -> Result<(), Box<dyn std::error::Error>>;
}
```

#### Behavior

Writes `content` to `path` via `std::fs::write`. Used by the editor (saving frontmatter edits) independently from creation.

## Scaffold Template

The generated `SKILL.md` has this structure:

```markdown
---
name: <name>
agents:
  - agent1
tags:
  - tag1
---

# <name>

[description]
```

The template body is generated by `core::scaffold_skill()`.

## Edge Cases

| Scenario | Behavior |
|---|---|
| Directory already exists | `std::fs::create_dir_all` succeeds (no-op) |
| Write fails (permissions) | I/O error propagated |
| Empty agents list | `agents` key omitted from frontmatter |
| Empty tags list | `tags` key omitted from frontmatter |

## Tests

- `create_returns_path_to_skill_directory`: verifies directory exists after create
- `create_writes_skill_md_in_base_dir`: verifies SKILL.md is created
- `created_skill_has_correct_frontmatter`: verifies name/agents/tags in output
- `fs_skill_writer_writes_content`: verifies file content matches
- `from_env_creates_in_home_dir`: `#[ignore = "requires ~/.claude/skills directory to exist"]`
