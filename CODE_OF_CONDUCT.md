# Code of Conduct

Este projeto adota uma expectativa simples: trate outras pessoas com respeito e mantenha discussões técnicas focadas no problema.

## Comportamento Esperado

- Seja direto sem ser hostil.
- Presuma boa intenção, mas valide afirmações com evidências.
- Aceite feedback técnico sem personalizar a crítica.
- Evite linguagem discriminatória, assédio, ameaças ou ataques pessoais.

## Aplicação

Mantenedores podem editar, ocultar ou remover comentários, issues e PRs que violem este código. Violações recorrentes podem resultar em bloqueio de participação no projeto.

## Reporte

Abra uma issue se o reporte puder ser público. Para casos sensíveis, use o canal indicado em `SECURITY.md` até haver um contato privado dedicado.
